# Campus Job Marketplace
College campuses are filled with students who possess diverse skills such as poster
designing, content writing, record preparation, tutoring, video editing, and more. However, there
is no structured system that connects these skilled individuals with students who require such
services. As a result, students depend on informal networks, which often lack reliability,
transparency, and accountability. This leads to missed opportunities—skilled students are unable
to effectively monetize their abilities, and service seekers struggle to find the right help at the
right time.

This project proposes a campus-exclusive Micro-Job Marketplace, a web-based platform that
enables students to offer and request small-scale services within the college environment. Users
can register using their college credentials, list their skills with descriptions and pricing, browse
or search for available services, and request jobs seamlessly. The system includes an admin
module for verification and moderation, ensuring safety and fairness for all participants. Features
such as order tracking, ratings, and reviews further enhance trust and reliability.

## Hardware Requirements
**Processor**: Intel i5 or higher

**RAM**: Minimum 8 GB

**Storage**: 256 GB or higher

**System**: Desktop or Laptop

## Software Requirements
**Operating System**: Windows / Linux

**Frontend**: React.js for UI

**Backend**: FastAPI for user authentication, job posting, job requests

**Database**: PostgreSQL for user profiles, job listings, applications

**Tools**: Visual Studio Code for development, Web Browser for testing, and Git for version control
